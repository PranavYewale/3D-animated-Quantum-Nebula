### Running the Server

1. **Install prerequisites**  
   Ensure that **Node.js** and **npm** (Node Package Manager) are installed on your computer.  
   - To verify, open a terminal/command prompt and run:  
     ```bash
     node -v
     npm -v
     ```
     Both commands should return version numbers if installed correctly.

2. **Navigate to the project folder**  
   Open **Command Prompt** (Windows) or **Terminal** (macOS/Linux), then move into your project directory using:  
   ```bash
   cd path\to\your\project
   ```

3. **Start the development server**  
   Run the following command:  
   ```bash
   npm run dev
   ```
